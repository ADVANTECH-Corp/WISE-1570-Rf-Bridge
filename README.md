# WISE-1570-Rf-Bridge

## Deployment

Please follow the below steps to compile this sample code:

1. `git clone https://github.com/ADVANTECH-Corp/WISE-1570-Rf-Bridge.git`

1. `cd WISE-1570-Rf-Bridge`

1. `mbed config root .`

1. `mbed deploy`

## Compilation

`mbed compile -m WISE_1570 -t GCC_ARM -c`
